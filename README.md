# AI Study Planner 📚🤖

An AI-powered study planning assistant built using **Python, Streamlit, and Groq API**. This application helps students organize their study schedule, track subject-wise progress, analyze study hours using visual charts, and interact with an AI assistant for personalized guidance.

## 🚀 Features

### 📅 Smart Study Planner
- Add and manage subjects
- Track dedicated study hours
- Organize your daily study routine
- Monitor your learning progress

### 📊 Data Visualization
- Generate charts based on study data
- Analyze time spent on different subjects
- Understand study patterns through visual insights

### 🤖 AI Chat Assistant
- Chat with an AI-powered study companion
- Get study suggestions and explanations
- Ask questions related to learning, planning, and productivity

### ⚡ Powered by Groq API
- Uses Groq's high-speed AI inference API
- Provides fast and intelligent responses through an integrated chatbot

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/AI-Study-Planner.git
````

### 2. Navigate to the project folder

```bash
cd AI-Study-Planner
```

### 3. Install required libraries

```bash
pip install -r requirements.txt
```

### 4. Add Groq API Key

Create a `.env` file:

```
GROQ_API_KEY=your_api_key_here
```

Replace `your_api_key_here` with your actual Groq API key.

---

## ▶️ Run the Application

Start the Streamlit app using:

```bash
streamlit run app.py
```

The application will open in your browser.
